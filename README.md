# Vault Desktop

Desktop password storage app. Local. Secure.

## Features

* Add/edit/delete note-pass values
* Visually hidden passwords
* Search by key word
* Copy to clipboard
* Log in/out by password
* Encrypted data
* Planned/unplanned backup of data
* Autorun
